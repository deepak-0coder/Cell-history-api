# 🔬 Cell-History-API
This project is a functional Python API that maps the **Structural and Functional** discovery of the cell from 1665 to 2024.

### 📅 Key Milestones Included:
* **1665:** Robert Hooke (Structural discovery)
* **1839:** Schleiden & Schwann (Functional Cell Theory)
* **2012:** CRISPR (Genetic Engineering)

### 💻 How it Works:
The script uses Python `dataclasses` to store discovery objects and provides methods to filter history by **Century** or **Scientist**.
